# 🌐 Kunal Kushwaha Portfolio Website

✨ A clean and simple **portfolio website inspired by Kunal Kushwaha**, built using **HTML5, CSS3, and Bootstrap 5**.  
This project presents important information such as **personal details, professional summary, and contact information** in a structured and visually appealing layout.

The main objective of this project is to demonstrate how a **modern portfolio website can be created using core web technologies along with Bootstrap 5 for styling and layout**. The website focuses on simplicity, readability, and clean UI design.


<br><br>


## 🚀 Overview

This portfolio website highlights the profile and background of **Kunal Kushwaha**.  
It is designed to present important information in a **clear and organized format**, making it easy for visitors to understand the profile quickly.

The website is a **lightweight static project**, which ensures fast loading and smooth performance. Bootstrap 5 is used to simplify the design process and provide responsive layout components.

This project can also serve as a **learning project for beginners who want to understand how HTML, CSS, and Bootstrap work together to build a portfolio website**.
